## What you can build

The Ramp Developer API gives you programmatic access to Ramp's full financial infrastructure. Whether you're building an ERP integration, automating card issuance, or syncing expense data, the API covers it all.

## Base URL

All API requests are made to:

```
https://api.ramp.com
```

## Authentication

The Ramp API uses **OAuth 2.0** for authentication. All requests must include a valid bearer token in the `Authorization` header.

```bash
Authorization: Bearer <your_access_token>
```

<Card title="Set up OAuth 2.0" icon="lock" href="/authentication/oauth2">
  Learn how to obtain and manage access tokens.
</Card>

## API Versioning

The current API version is **v1**, reflected in all endpoint paths:

```
https://api.ramp.com/developer/v1/...
```

## OpenAPI Spec

The full OpenAPI specification is available for download and use with tools like Postman, Insomnia, or code generators:

```
https://docs.ramp.com/openapi/developer-api.json
```
